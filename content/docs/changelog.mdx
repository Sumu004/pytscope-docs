## [0.2.1] — 2026-06-08

Renamed the project from `trainscope` to `pytscope` (`trainscope` and
`tscope` were already taken on PyPI). Import path, CLI command, PyPI
distribution, and GitHub repo are all `pytscope` now — no functional changes.

## [0.2.0] — 2026-06-08

Amber-LED "hardware panel" terminal reports, a Training Efficiency Budget
anchored to MFU, exposed-communication analysis from `torch.profiler` traces,
zero-touch `AutoProfiler` instrumentation, and a full distributed vertical
(straggler detection, load imbalance, pipeline bubbles). Validated on a real
multi-GPU NCCL run (Kaggle 2× T4) — see [Validation](/docs/validation). Also
fixed a Windows CLI crash on unicode output and an MFU device-selection bug.

## [0.1.0] — 2026-06-06

First public beta: the `Profiler` core, one telemetry backbone feeding four
analysis verticals (timing, memory, convergence, cross-signal) plus
reproducibility diffing, Lightning/Hugging Face integrations, and the
`analyze`/`diff` CLI.
