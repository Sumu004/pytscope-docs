import { Card, Cards } from 'fumadocs-ui/components/card';

Every signal pytscope captures lands on the same per-step timeline
(`StepRecord`s in a `RunStore`). Each analyzer reads that one stream and
returns a typed summary; the diagnosis engine turns summaries into ranked
`Finding`s — each with a severity, a plain-language explanation, and a
concrete fix.
